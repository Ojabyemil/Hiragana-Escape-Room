# Hiragana

This project is made for the stardance challenge by hack club.

This is a very simple console game using the C# console.
It is mainly made for those who cannot read hiragana since you can skip all puzzles by just reading the hiragana. 

This was an idea I got during my last programming lesson of the semester, this project was not started until June 6th (the day I'm writing this) so all code has been written during this event.
